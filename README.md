# Tourist Management Dashboard (Qt)

## Project Overview

This project is a **Tourist Management Dashboard** developed using **Qt and C++** as part of my learning process. The goal of this project was to explore the Qt framework and understand how to build GUI-based desktop applications.

The dashboard contains multiple buttons that open different forms related to tourist management features.

## Features

* Dashboard interface built with Qt
* Navigation buttons for different operations
* Forms for basic tourist management tasks
* Multi-window navigation using Qt Widgets

## Learning Objectives

While building this project, I learned:

* Basics of **Qt Creator**
* Creating **GUI applications using Qt Widgets**
* Working with **.ui design files**
* Connecting buttons with functionality using **signals and slots**
* Organizing code using **C++ classes and header files**

## Technologies Used

* C++
* Qt Framework
* Qt Creator

## Project Status

This project is currently a **learning prototype** created to understand Qt development. Some features are still basic and can be expanded further.
